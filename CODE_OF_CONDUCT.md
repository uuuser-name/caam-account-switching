# Contributor Covenant Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone, regardless of:

- Identity and background
- Experience level
- Gender, race, religion, or politics
- Technical preferences or tooling stack

## Our Standards

Examples of behavior that helps this project stay healthy:

- Use respectful language
- Be concise, collaborative, and evidence-based
- Focus on code and behavior, not the person
- Accept review feedback in good faith
- Report security-sensitive issues privately when appropriate

Examples of unacceptable behavior:

- Personal attacks or harassment
- Threats, stalking, or doxxing
- Repeatedly dismissive behavior in code review
- Sharing credentials, secrets, or token material

## Scope and enforcement

This code applies to all project channels: issues, pull requests, and any official project discussion.

Project maintainers may:

- Ask for behavior correction
- Temporarily restrict participation
- Close or archive abusive threads
- Ban repeat offenders

## Reporting

If you experience or witness a violation, open an issue with a concise report and proof if possible.
For urgent security/privacy violations, report privately using the security process in [SECURITY.md](SECURITY.md).
